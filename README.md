# Gerenciador de Banco de Dados Pessoal e Evolução Patrimonial 📈

Sistema desenvolvido em Python para controle, armazenamento e acompanhamento de evolução patrimonial. O projeto utiliza banco de dados local para persistência e gera análises visuais dos investimentos.

## 🚀 Tecnologias Utilizadas
* *Python* (Lógica principal do sistema)
* *SQLite3* (Gerenciamento do banco de dados relacional local)
* *Matplotlib* (Renderização e plotagem do gráfico de evolução)
* *Datetime* (Manipulação e registro automático de datas)
